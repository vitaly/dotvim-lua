## File Tree

[neotree](https://github.com/nvim-neo-tree/neo-tree.nvim)

### Config

- `my.config.neotree.position` - left (default), or right

## Maps

- `\\\\` - Quick File Popup
- `\\[` - Toggle File Tree
- `\\[` - Toggle File Tree / Reveal File

inside File TRee:

- `?` - show help
- `a` - add file / dir
- `r` - rename file / dir
- `m` - move file / dir
- `d` - delete file / dir
- `q` - close File Tree

## Basic Keymaps

- `<tab><tab>` - Last Tab
- `<tab>n` - New Tab

- `<SPACE>m` - show messages
- `<SPACE>M` - clear messages

- `<SPACE>x` - quite window
- `<SPACE>qq` - quite all
- `<SPACE>qQ` - quite all!
- `<SPACE>Q` - quite all!

- `<SPACE>n` - Next
- `<SPACE>p` - Prev

- `<SPACE>a` - Show Apps Menu
- `<SPACE>b` - Show Buffer Menu
- `<SPACE>f` - Show File Menu
- `<SPACE>w` - Show Window Menu
- `<SPACE>s` - Show Status Menu
- `<SPACE>t` - Show Tab Menu
- `<SPACE>v` - Show Vim Menu
- `<SPACE>Y` - Show Yank Menu

- `,<SPACE>` - Save
- `,e` - Show Eval Menu

- `\` - Show Toggle Menu

- `\l` - Show Log Menu
- `\lt` - Log TRACE
- `\ld` - Log DEBUG
- `\li` - Log INFO
- `\lw` - Log WARN
- `\le` - Log ERROR
- `\lo` - Log OFF

### in diff-mode

- `,d` - Show Diff Menu

## Basic Keymaps

- `<tab><tab>` - Last Tab
- `<tab><space>` - New Tab

- `<space>m` - show messages
- `<space>M` - clear messages

- `<space>x` - quite window
- `<space>qq` - quite all
- `<space>qQ` - quite all!
- `<space>Q` - quite all!

- `<space>n` - Next
- `<space>p` - Prev

- `<space>a` - Show Apps Menu
- `<space>b` - Show Buffer Menu
- `<space>f` - Show File Menu
- `<space>w` - Show Window Menu
- `<space>s` - Show Status Menu
- `<space>t` - Show Tab Menu
- `<space>v` - Show Vim Menu
- `<space>Y` - Show Yank Menu

- `<space><space>` - Save
- `,e` - Show Eval Menu

- `\` - Show Toggle Menu

- `\l` - Show Log Menu
- `\lt` - Log TRACE
- `\ld` - Log DEBUG
- `\li` - Log INFO
- `\lw` - Log WARN
- `\le` - Log ERROR
- `\lo` - Log OFF

### in diff-mode

- `,d` - Show Diff Menu

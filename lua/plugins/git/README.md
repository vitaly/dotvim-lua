## Git

[gitsigns.nvim](https://github.com/lewis6991/gitsigns.nvim)
[vim-fugitive](https://github.com/tpope/vim-fugitive)

### Gitsigns

| Key | Description |
|-----|-------------|
| `\gs` | Toggle Signs |
| `\gv` | Toggle Virtual Line Blame |
| `\gl` | Toggle Line Highlight |
| `\gw` | Toggle Word Diff |

### Fugitive

| Key | Description |
|-----|-------------|
| `<leader>gs` | Git Status |
| `<leader>gd` | Git Diff (vertical) |
| `<leader>gp` | Git Push |
| `<leader>gg` | Git Grep... |
| `<leader>gw` | Git Grep Word |

### Git Status Mode

| Key | Description |
|-----|-------------|
| `g?` | Help |

**Changes:**

| Key | Description |
|-----|-------------|
| `-` | Stage/unstage file or hunk |
| `X` | Discard changes |

**Commit:**

| Key | Description |
|-----|-------------|
| `cc` | Commit |
| `ca` | Amend commit |
| `cf` | Fixup commit |
| `cF` | Fixup and rebase |

**Rebase:**

| Key | Description |
|-----|-------------|
| `ri` | Interactive rebase |
| `rr` | Continue |
| `rs` | Skip |
| `ra` | Abort |
| `rw` | Reword |
| `rm` | Edit |
| `rd` | Drop |

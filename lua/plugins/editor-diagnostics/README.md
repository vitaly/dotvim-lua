# editor-diagnostics

[trouble.nvim](https://github.com/folke/trouble.nvim) — pretty list for diagnostics, quickfix, and location list.

## Keybindings

| Key | Action |
|-----|--------|
| `\DD` | Toggle workspace diagnostics |
| `\Db` | Toggle buffer diagnostics |
| `\Dq` | Toggle quickfix list |
| `\Dl` | Toggle location list |
| `\D\` | Toggle diagnostics |
| `[d` / `]d` | Prev / Next diagnostic |
| `[D` / `]D` | First / Last diagnostic |
| `[e` / `]e` | Prev / Next error |
